Welcome to Swindon’s documentation!¶

Swindon is a web server that eventually should develop all the features needed for a frontend server. But the most powerful feature is a protocol for handling websockets.

Github | Crate

While swindon is quite recent project it handles about 120k simultaneous connections and ~20 million messages per day in our setup. The screenshot above shows just a random day from our dashboard.
